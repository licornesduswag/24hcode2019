Release Notes for X-CUBE-VS4A Firmware Package

Copyright 2018 STMicroelectronics

 


X-CUBE-VS4A provides STVS4A and dependencies. It is STMicroelectronics middleware that enables a customer to implement an Alexa Voice Service enabled product.

The document UM2354, available on st.com, gives more information about the package content.
